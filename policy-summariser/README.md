# Policy Plain-English Summariser
Turns complex insurance policy documents into clear, plain-English summaries any adult can understand.

## What it does
- Extracts what is covered and what is not
- Surfaces key numbers (deductible, limits, premium)
- Gives a simple 3-step claims process
- Flags the one thing most people miss

## Tools used
- Claude.ai (Projects + system prompt)
- No code required

## Status
- System prompt: done
- Test output: done

## ML concept learned
Tokenisation and context windows models read text in chunks. 
Long policy documents can exceed limits, so knowing what to prioritise in the prompt matters.
